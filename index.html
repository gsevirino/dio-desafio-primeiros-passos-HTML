O que é HTML?

HTML é a sigla para HyperText Markup Language (Linguagem de Marcação de Hipertexto). É uma linguagem de marcação utilizada para criar a estrutura e o conteúdo de páginas da web. É a base fundamental da maioria dos sites na Internet. HTML não é linguagem de programação!

Embora o HTML seja responsável pela estrutura e o conteúdo de uma página, ele não lida com a parte visual. Para estilizar as páginas, é comum utilizar o CSS (Cascading Style Sheets), que controla o layout, as cores, as fontes e outros aspectos visuais.

Tags do HTML

O HTML utiliza uma sintaxe composta por TAGS, que são códigos inseridos em um documento para definir a forma como o conteúdo deve ser exibido no navegador. Cada tag tem um propósito específico e determina como um elemento deve ser exibido na página.

As tags HTML podem ser usadas para criar cabeçalhos, parágrafos, listas, links, imagens, tabelas e muitos outros elementos encontrados em páginas da web. Além disso, o HTML também permite incorporar conteúdo multimídia, como vídeos e áudios, em um documento.

Atributos das Tags

Atributos são informações adicionais fornecidas nas tags HTML que modificam o comportamento ou a aparência dos elementos. Os atributos são usados para fornecer instruções sobre como as Tags devem ser exibidas ou se comportar.

Os atributos são inseridos dentro das tags como pares chave-valor, separados por um sinal de igual (=), atributo="valor". Algumas Tags podem ter atributos obrigatórios, enquanto outros têm atributos opcionais.

Alguns exemplos de atributos comuns em HTML:

  • class: Define uma classe CSS para a tag.
  • id: Define um identificador exclusivo para a tag.
  • style: Define estilos CSS para a tag.
  • src: Especifica o caminho ou URL de uma imagem, vídeo ou outro recurso.
  • href: Especifica o destino de um link (âncora) ou a localização de um arquivo externo.
  • alt: Fornece um texto alternativo para imagens, exibido caso a imagem não possa ser carregada.
  • width e height: Especificam as dimensões de uma imagem ou de umaa tag.
